Background Images, Amigos

Properties

There are few different background properties:

background-image

This property is the location of the image itself, for example:

h1 { background-image: url ('http://website.com/img/background.jpg');}

or

h1 { background-image: url('image/background.jpg');}

background-color

This allows us to add color to a background. An example in CSS would look like this:

h1 { background-color: green; }

background-repeat

This is how the image repeat. The value can be any of the folowing

background-repeat: repeat;

This repeats on both the X and Y axis, and will essentially title the image across the entire background.
